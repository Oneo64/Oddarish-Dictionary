Adjective Conjugations

I'm too lazy to code this in, so here's a big table on adjective conjugations.

Some basic neuter rules:

  1. -ðr and -tr becomes -tt (rauðr -> rautt, svétr -> svétt)
  2. -dr becomes -t (kaldr -> kalt, sundr -> sunt)
  3. -gr, -kr, -mr, and -nr adds a -t (dimmr -> dimmt, falligr -> falligt)
  4. -inn and -ill becomes -ið (ávokninn -> ávoknið, lýtill -> lýtið)

Other rules:

  1. -inn and -ill becomes -n + vowel and -l + vowel (feiminn -> feimnu, lýtill -> lýtlu)
  2. -vinn becomes -ø + vowel or -u + ending (statvinn -> statvu/statvi/statva, statvinn -> staturan)
  3. -einn becomes -eið + ending (hreinn -> hreiðu/hreiði/hreiða, hreinn -> hreiðran)
  4. adjectives ending in -andi and other irregular endings are not declensible
